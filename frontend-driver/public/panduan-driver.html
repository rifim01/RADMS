RADMS DRIVER APP

Panduan Driver RIFIM Airport

Langkah lengkap dari install hingga selesai perjalanan

Bagikan panduan ini: WhatsApp
Alur penggunaan — 7 langkah
1

Install aplikasi di HP

PWA — tidak perlu Play Store / App Store

  • Buka Chrome di Android → kunjungi link aplikasi dari koordinator
  • Ketuk menu ⋮ pojok kanan atas → pilih "Tambahkan ke Layar Utama"
  • Ketuk Tambah — ikon RIFIM muncul di homescreen HP
  • iPhone/Safari: ketuk Share → "Add to Home Screen"
  • Izinkan notifikasi saat browser meminta
ℹ️ Info: Aplikasi bisa digunakan tanpa koneksi internet setelah pertama kali dibuka (mode offline terbatas).
2

Login dengan NIK & Nama

Gunakan data terdaftar di sistem RIFIM

  • ID Driver = NIK Anda (angka saja, contoh: 158756364)
  • Nama Lengkap = nama persis sesuai data RIFIM (minimal kata pertama)
  • Ketuk tombol merah Masuk
  • Berhasil → masuk ke halaman utama aplikasi
⚠️ Penting: 1 akun hanya bisa aktif di 1 HP. Login di HP lain otomatis mengakhiri sesi HP pertama.
3

Aktifkan status Online

GPS aktif — posisi terpantau koordinator

  • Di halaman Home, ketuk toggle Online/Offline atau tombol Mulai Bertugas
  • Status berubah menjadi ● ONLINE — GPS Anda aktif
  • Izinkan akses lokasi jika browser meminta permission
  • Posisi terakhir Anda tetap tersimpan meski aplikasi ditutup
📡 GPS otomatis: Tracking berjalan di background — tidak perlu buka aplikasi terus menerus.
4

Masuk area bandara & antrian

Sistem otomatis mendeteksi kedatangan Anda

  • Saat memasuki radius 2 km dari bandara — notifikasi Geofence Aktif muncul
  • Ketuk tombol Masuk Antrian di halaman Home atau menu Antrian
  • Nomor antrian Anda ditampilkan — tunggu giliran dipanggil
  • Harus berada dalam radius bandara agar tombol antrian aktif
● Menunggu ● Dipanggil ● Selesai
5

Terima panggilan dari staff

Notifikasi suara dan layar saat giliran tiba

  • Aplikasi berbunyi notifikasi dan menampilkan "Anda Dipanggil!"
  • Status antrian berubah menjadi ● Dipanggil
  • Segera menuju zona penjemputan yang ditentukan koordinator
  • Staff akan konfirmasi kehadiran Anda di titik jemput
🔔 Suara aktif: Pastikan volume HP tidak silent. Izin notifikasi browser harus diberikan saat pertama install.
6

Selesaikan perjalanan

Rekam trip setelah mengantar penumpang

  • Setelah mengantar penumpang, kembali ke aplikasi
  • Ketuk tombol Selesai / Complete Trip
  • Riwayat perjalanan otomatis tersimpan di menu Riwayat
  • Anda bisa langsung masuk antrian lagi untuk penjemputan berikutnya
📋 Riwayat tercatat: Koordinator dapat memverifikasi semua perjalanan dari dashboard staff.
7

Tombol darurat (Panic Button)

Hanya gunakan dalam kondisi darurat nyata

  • Tekan dan tahan tombol ● DARURAT di halaman utama
  • Sistem mengirim sinyal darurat + lokasi GPS ke koordinator secara instan
  • Bantuan akan segera dikirimkan ke lokasi Anda
  • Tombol cooldown 60 detik untuk mencegah pengiriman berulang
⚠️ Hanya untuk darurat nyata. Penyalahgunaan tombol ini akan tercatat dan dilaporkan ke manajemen.
Masalah umum & solusi
❌ Tidak bisa login — NIK tidak ditemukan
Pastikan NIK yang dimasukkan hanya angka dan sesuai data RIFIM. Hubungi koordinator untuk verifikasi NIK Anda di sistem.
🔄 Tiba-tiba keluar dari aplikasi sendiri
Akun digunakan di HP lain. Login ulang — sesi HP lain otomatis berakhir. Jangan bagikan akun ke driver lain.
📍 Tombol "Masuk Antrian" tidak aktif / abu-abu
Pastikan GPS HP aktif (mode Akurasi Tinggi) dan status Anda sudah Online. Anda harus berada dalam radius 2 km dari bandara.
🔕 Tidak menerima notifikasi suara panggilan
Pastikan volume HP tidak silent. Aktifkan izin notifikasi: Pengaturan → Aplikasi → Chrome → Notifikasi → Aktifkan.
🌐 GPS tidak akurat atau tidak terdeteksi
Aktifkan mode Akurasi Tinggi di pengaturan lokasi HP. Keluar dari dalam gedung jika sinyal lemah. Izinkan akses lokasi saat browser meminta.
📶 Aplikasi error saat tidak ada internet
Aplikasi dapat digunakan dalam mode terbatas tanpa internet. Koneksi diperlukan untuk masuk antrian dan update lokasi real-time.
